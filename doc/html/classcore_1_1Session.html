Descrição Detalhada

Classe para controle da sessão do usuário.

Esta classe controla o estado do sistema. Nela ficam armazenados ponteiros para o usuário logado, para a classe de equipe e para a classe de estoque.

Construtor que define os dados inicias da sessão.

Aqui são inicializados os dados de estoque e da equipe a partir dos arquivos .data. O _currentUser é inicializado com nullptr.
